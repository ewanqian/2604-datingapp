## 🎭 这是什么？

最近悄悄做了一个 app  
它是一个面向当代艺术、数字媒体、新媒体影像、web3、空间创作及相关文化行业的匹配平台。

不是普通社交软件那种逻辑，也不只是为了"认识人"。

更准确一点说，它在测试一种新的匹配方式：  
除了兴趣、审美和聊天氛围之外，  
人与人之间是否也可以基于语境相容度、合作潜力、行业路径、可见性和长期价值来建立连接。

这两年越来越觉得，很多关系表面上是偶遇，  
本质上其实是一种更复杂的相互选择。  
你选择的未必只是一个对象，  
也可能是一种方法、一种圈层位置、一种新的自我叙述，  
甚至是一条通往别的场域的路径。

所以这更像是一个小型实验：  
看看当关系、合作、身份与行业流动混在一起的时候，  
一个匹配系统会长成什么样。

12+ 位精心设计的候选人，每一位都精准戳中了当代艺术圈的某种生态位——

- 标签越来越多，职业越来越糊，但名片越来越厚
- 研究很前沿，自拍也很前沿，生活像永远在做 identity launch
- 最会的不是创作，是把任何事说成生态
- 水平一般，但非常会出现在正确的地方
- 会把一切问题都讲得高级，也会把一切人都看成案例
- 真的会一些技术，但更会把 demo 讲成愿景

滑动，匹配，感受一下你在艺术圈的"语境相容度"。

## 🤝 如何提交自己

不懂 GitHub 也没关系，直接在网页里提交：

1. 打开 [CURATED](https://ewanqian.github.io/2604-datingapp/curator_app_prank.html)
2. 点「提交我自己」
3. 填写你的信息，点击推荐标签快速添加
4. 点「提交到 GitHub Issue」→ 自动帮你打开 GitHub，信息已经预填好了
5. 上传你的 2:3 压缩肖像 → 点「Create new issue」完成
6. 留下联系方式方便别人找到你

### 🔄 自动化处理流程

提交 Issue 后，系统会**全自动处理**：

1. **GitHub Action** 自动提取信息、下载图片、添加代码、创建 Pull Request
2. 如果 Action 失败，**AI Agent**（`auto-github-profile-submission` skill）会手动处理
3. 每天定时自动刷新，维护者只需要审核合并即可
4. 合并后几分钟，你就能在在线体验页看到自己了

### 完全开源自动化

这个项目使用 GitHub + AI Agent 完全自动化运营：

- 用户 → 网页表单 → GitHub Issue
- GitHub Action → 自动处理 → Pull Request
- AI Agent → 自动兜底处理失败的任务
- 维护者 → 审核 → 一键合并 → 完成

详细贡献指南请看 [CONTRIBUTING.md](CONTRIBUTING.md)

## 🔍 如何联系别人

如果你对某个人感兴趣，想联系他/她：

1. 记住这个人的名字
2. 去 [Issues](https://github.com/ewanqian/2604-datingapp/issues) 搜索他/她的名字
3. 在 Issue 里就能找到联系方式，直接留言就好

所有人提交的时候都会被提示留下联系方式，所有人都能看到。

## 📝 格式要求

提交需要：

- 姓名 / 年龄 / 性别
- 身份：多重身份标签（用 / 分隔）
- 讽刺点：一句话精准定位你的生态位
- 肖像图片：2:3 竖屏比例，JPEG 格式，压缩到 10-20KB 左右
- 3-4 个标签（网页有常见标签快速点击）
- （可选）联系方式方便别人找到你
