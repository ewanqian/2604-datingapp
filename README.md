# CURATOR

一个伪装成社交产品的关系型网页实验。

CURATOR 以移动端匹配界面为壳，结合角色资料、聊天、相册、动态与活动轨迹，模拟一种既亲密又充满误读的关系体验。  
它表面上像一个精致的小众社交软件，实际上更接近一个单人可玩的角色关系模拟器：用户不是简单地“配对”角色，而是在聊天、查看资料、解锁图片与触发事件的过程中，慢慢认识一个人，也慢慢暴露自己会如何靠近、试探、使坏、撤退或继续停留。
