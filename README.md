# CURATOR

一个伪装成社交产品的关系型网页实验。

CURATOR 以移动端匹配界面为壳，结合角色资料、聊天、相册、moments、活动轨迹与后续可扩展的黑色幽默分支，模拟一种既亲密又充满误读的关系体验。

它表面上像一个精致的小众社交软件，实际上更接近一个单人可玩的角色关系模拟器：  
用户不是简单地“匹配”角色，而是在聊天、查看资料、解锁图片与触发事件的过程中，慢慢认识一个人，也慢慢暴露自己会如何靠近、试探、使坏、撤退或继续停留。

## 当前重点
1. 修正通用语料过于自来熟的问题
2. 让不同角色的开场更有区分度
3. 完整接入安由纪 / Yuki
4. 让资料页、聊天、相册与 moments 逐步联动

## 当前设计原则
- 不做真实社交平台
- 重点放在角色魅力、图像、关系张力与长期文本体验
- 文本要有生活感，但不能一上来就像认识很久
- 资料页、聊天、相册与事件应互相推动
