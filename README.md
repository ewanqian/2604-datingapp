## 🎭 这是什么？

这是一个愚人节玩笑/行为艺术，模拟了一个专为文化艺术圈人士打造的"高端"约会App。

12 位精心设计的候选人，每一位都精准戳中了当代艺术圈的某种生态位——

- 标签越来越多，职业越来越糊，但名片越来越厚
- 研究很前沿，自拍也很前沿，生活像永远在做 identity launch
- 最会的不是创作，是把任何事说成生态
- 水平一般，但非常会出现在正确的地方
- 会把一切问题都讲得高级，也会把一切人都看成案例
- 真的会一些技术，但更会把 demo 讲成愿景

滑动，匹配，感受一下你在艺术圈的"匹配度"。

## 🤝 欢迎提交自己！

我们欢迎任何人提交自己的 profile 加入这个项目！

如果你觉得自己也符合某种"艺术圈生态位"，或者想创作一个新的讽刺角色，欢迎：

1. Fork 这个仓库
2. 在 `curator_app_prank.html` 的 `profiles` 数组中添加你的信息
3. 在 `pics/` 目录添加你的 2:3 比例肖像（质量 60% 左右的 JPEG）
4. 提一个 Pull Request 或者开一个 Issue

详细指南请看 [CONTRIBUTING.md](CONTRIBUTING.md)

## 📝 格式要求

添加新 profile 需要：

- 姓名 / 年龄 / 性别
- 身份：你的多重身份标签
- 讽刺点：一句话精准定位
- 肖像图片：2:3 比例，JPEG 格式，压缩到 10-20KB 左右
- 三到四个标签
